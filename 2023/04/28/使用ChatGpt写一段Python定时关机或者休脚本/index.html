使用ChatGpt写一段Python定时关机脚本

相信大多数人有如下的需求：

  • 需要在特定的时间点进行关机或者休眠，比如你在下载一个大文件，耗时比较久，想在23：00自动关机
  • 打开特定APP想30分钟后自动关机或者休眠，比如家里有熊娃娃使用优酷看动画片，你想每次打开优酷的时候，30分钟后自动关机。

于是乎，我就让ChatGpt帮我实现以上需求了。

我想让你充当专家级别的软件开发人员。您只会回复我你认为最为标准权威的答案。请注意，您的回答应该是简明扼要的，在需要的时候你可以回复表格，千万记住不需要附带任何额外的解释。首先请回复：我现在需要一个python脚本。需求如下：
1.目标是定时关机
2.可以在运行的时候设置具体时间关机，如：python shutdown.py  22:10 就是22点10分准时关机。
3.可以设置倒计时关机，如：python shutdown.py 30 就是30分钟之后关机
4.在控制台每秒打印日志，还有多少秒即将关机
import os
import sys
import time

def shutdown_timer(time_arg):
    if ':' in time_arg:
        # 设置具体时间关机
        shutdown_time = time_arg.split(':')
        shutdown_hour = int(shutdown_time[0])
        shutdown_minute = int(shutdown_time[1])
        current_time = time.localtime()
        current_hour = current_time.tm_hour
        current_minute = current_time.tm_min
        if shutdown_hour < current_hour or (shutdown_hour == current_hour and shutdown_minute <= current_minute):
            shutdown_hour += 24
        shutdown_seconds = (shutdown_hour - current_hour) * 3600 + (shutdown_minute - current_minute) * 60
    else:
        # 设置倒计时关机
        shutdown_seconds = int(time_arg) * 60

    for i in range(shutdown_seconds, 0, -1):
        print(f"{i} seconds until shutdown.")
        time.sleep(1)

    os.system("shutdown /s /t 1")

if __name__ == '__main__':
    if len(sys.argv) == 2:
        shutdown_timer(sys.argv[1])
    elif len(sys.argv) == 3:
        shutdown_time = sys.argv[2]
        shutdown_timer(shutdown_time)
    else:
        print("Usage: python shutdown.py [minutes|hh:mm]")

ChatGpt给的脚本可以直接运行，并且经过我的验证可以直接使用

你可以在命令行中运行此脚本，例如：

python shutdown.py 22:10
python shutdown.py 30

脚本将在控制台每秒打印日志，显示还有多少秒即将关机。在指定的时间或倒计时结束后，脚本将执行关机命令。
